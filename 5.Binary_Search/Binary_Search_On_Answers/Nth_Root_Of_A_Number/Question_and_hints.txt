Nth Root Of A Number
---------------------
Problem Statement: Given two numbers 'n' and 'x', find the nth root of 'x'. The nth root of a number 'x' is 
defined as a number 'r' when raised to the power 'n' equals 'x'. If the nth root is not an integer, return -1.


Example 1:
Input Format:
 n = 3, x = 27
Result:
 3
Explanation:
 The cube root of 27 is equal to 3.


Example 2:
Input Format:
 n = 4, x = 69
Result:
 -1
Explanation:
 The 4th root of 69 does not exist. So, the answer is -1.
